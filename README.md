# MileLog

An iOS app for tracking vehicle mileage for tax deductions.

## Design

**Liquid Glass UI** - A modern glassmorphism design featuring:
- Frosted glass backgrounds with blur effects
- Translucent panels with subtle transparency
- Soft shadows and rounded corners
- Vibrant colors showing through glass layers
- Smooth animations and transitions

## Features

- Log trips with date, odometer readings, and distance
- **Swipe to classify**: Swipe right for Business, swipe left for Personal
- Haptic feedback for confirmation
- Calculate tax deductions using IRS mileage rates
- View trip history and summaries
- Export reports for tax filing

## Requirements

- iOS 26.0+
- Xcode 16.0+

## Getting Started

1. Open `MileLog.xcodeproj` in Xcode
2. Select an iPhone simulator or device
3. Build and run the project (Cmd + R)

## Usage

1. Tap **+** to add a new trip
2. Enter start/end odometer readings or manual distance
3. Swipe right on a trip to mark as **Business** (green)
4. Swipe left on a trip to mark as **Personal** (blue)
5. View your total deductible miles in the summary
6. Export reports at tax time

## IRS Mileage Rates (2024)

- Business: 67 cents per mile
- Medical/Moving: 21 cents per mile
- Charity: 14 cents per mile
